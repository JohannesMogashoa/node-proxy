# Node Proxy Server

A simple proxy server built using Express JS built following a YouTube tutorial.

<h6>Env Variables</h6>

<ul>
    <li> API_BASE_URL = "https://api.openweathermap.org/data/2.5/weather"
    <li> API_KEY_NAME = "appid"
    <li> API_KEY_VALUE = "Own API KEY"
</ul>
